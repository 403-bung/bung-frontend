## 30분 안에 시작하는 벙개모임, 벙 (BUNG)

벙은 30분 안에 무언가를 같이 할 사람을 바로 구할 수 있는 대학생 커뮤니티로, 게임, 공구 등 지금 무언가를 같이 할 팀원들을 빠르게 찾을 수 있어요.
